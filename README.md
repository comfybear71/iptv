# ComfyTV

Premium IPTV subscription storefront for the ComfyTV crew. Built with Next.js 14, MongoDB, NextAuth (Google), and Tailwind CSS. Pay with crypto (SOL or BUDJU).

## Getting Started

```bash
npm install
cp .env.example .env   # Fill in your env vars
npm run dev
```

See [CLAUDE.md](./CLAUDE.md) for full documentation.
